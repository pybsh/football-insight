### Introduction
A website where you can check international football match schedules and participate in voting to predict match results.<br/>

### Features
- Login / Sign Up
- International football match schedule viewing
- Past match results viewing
- Match result prediction voting

## Other
- Match content is hard-coded by crawling from [Daum Sports](https://sports.daum.net/).
