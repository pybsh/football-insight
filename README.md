### 소개
해외 축구 경기 일정을 확인하고, 경기 결과를 예측하는 투표를 진행해 볼 수 있는 웹사이트입니다.<br/>

### 기능
- 로그인 / 회원가입
- 해외 축구 경기 일정 조회 기능
- 과거 경기 결과 조회 기능
- 경기 결과 예측 투표 기능

## 기타
- 경기에 대한 내용은 [다음 스포츠](https://sports.daum.net/)에서 크롤링해 하드 코딩되어있습니다.
